Mavora Privacy Policy

Last updated: May 21, 2026

Mavora is an anonymous realtime chat app.

Data We Process

Mavora uses Firebase Anonymous Authentication to create an anonymous app user ID. The app stores your alias, battery percentage, charging state, online presence, last seen timestamp, chat request metadata, chat membership metadata, and message text for active conversations.

Battery-Based Discovery

Mavora uses your current battery percentage to show other online users at the same percentage. Battery data is used for realtime matching and chat send-lock behavior.

Messages And Retention

Messages are shown only for the last hour. Clients opportunistically delete expired messages when Home or Chat is opened or before sending. Channel summaries may remain locally visible longer: 24 hours for Free and 7 days for Mavora Plus.

Notifications

Mavora can use Firebase Cloud Messaging for message and request notifications. Push tokens are sent to Mavora's notification worker and are not stored in public Realtime Database user profiles. Notification text does not include message content.

Payments

Mavora Plus uses RevenueCat and Google Play Billing when subscriptions are enabled. RevenueCat uses the anonymous Firebase user ID as the purchase user ID so purchases can be restored for the same anonymous app identity.

Deletion Requests

You can request deletion of your anonymous app data by contacting support with your in-app alias and approximate last usage time. Because Mavora does not require an email account, this information helps identify the correct anonymous profile.
